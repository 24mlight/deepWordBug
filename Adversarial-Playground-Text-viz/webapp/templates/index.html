Adversarial Playground Text

About

In recent years, Deep Neural Networks have become an essential tool for many machine learning tasks, especially image classification. However, this has given rise to the study of adversarial samples--carefully crafted inputs for a neural network that result in an incorrect classification.

In the spirit of the TensorFlow Playground, we present the Adversarial Playground Text: a web-based visualization tool to assist users in understanding the differences between common adversarial machine learning techniques. This application is based off of a previous project from the QData group: Andrew Norton's Adversarial Playground, however this application focuses on adversarial attacks on text.


About the Attacks


DeepWordBug Algorithm
The core idea of this algorithm is to generate efficient adversarial samples on text input. The algorithm can attack both Char and Word model in a fast and black-box manner. For most effective usage, please input a text sequence of at least 5 words.

Notable features of the DeepWordBug are the model, power, transformer algorithm, and scoring algorithm.

  • Power:

    The number of words per sequence that will be modified. If this number is higher than the length of the text sequence, then all the words in the sequence will be modified.

  • Scoring Algorithm:

    (1) Combined: Combination of the next two options Combined_score(x) = THS(x) + Î»(TTS(x))
    (2) Temporal: aka Temporal Head Score (THS),
    (3) Tail: aka Temporal Tail Score (TTS),
    (4) Replaceone: Score of the .
    (5) Random (baseline): This scoring function randomly selects tokens as targets. In other words, it has no method to determine which tokens to attack.
    (6) Gradient (baseline): Contrary to random selection which uses no knowledge of the model, we also compare to full knowledge of the model, where gradients are used to find the most important tokens.

  • Transformer Algorithm:

    (1) Swap: Swap two adjacent letters in the word.
    (2) Substitution: Substitute a letter in the word with a random letter.
    (3) Deletion: Delete a random letter from the word.
    (4) Insertion: Insert a random letter in the word.
